## AutoReliant – Test Automation & Reliability Platform

Built an automated API testing framework using pytest.

### Features
- Automated API testing
- Response validation
- Re-usable API client architecture
- HTML test reporting

### Run tests

pytest

### Generate report

pytest --html=reports/report.html
